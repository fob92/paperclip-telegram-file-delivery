# Production Pack

This skill package now includes:

- production delivery code
- UI install checklist
- import contract
- rollout and guardrails docs
- company addendum templates
- agent addendum templates
- env example for operator handoff
- validation script and CI workflow

## Minimal productive install

1. Install the skill through the Paperclip UI.
2. Attach it to the relevant company or delivery-capable agents.
3. Merge the company template guidance into company instructions.
4. Merge the delivery-agent template into the executing agent.
5. Add Telegram env values to the deployment `.env`.
6. Run the smoke tests.
