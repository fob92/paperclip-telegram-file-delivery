# v1.0.1

## Critical packaging fix

- restructured the repository to follow the working Paperclip-compatible layout:
  - `skills/telegram-file-delivery/SKILL.md`
  - sibling `scripts/`, `references/`, and `assets/`
- added self-contained `scripts/telegram_delivery.py` for installed runtime use
- duplicated operational docs/templates/examples into `references/` so they survive Paperclip skill import more reliably
- added a packaged-script smoke test

## Why this release exists

A prior root-level layout led to UI installs that surfaced `SKILL.md` without the operational helper files. This release is specifically aimed at Paperclip UI install compatibility.
